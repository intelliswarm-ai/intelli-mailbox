WHAT YOU ARE INSTALLING

Intelli Mailbox sits on top of your existing Gmail and uses an
AI to triage every email before you read it. Each message
arrives tagged, summarised, and (when relevant) with three
pre-drafted replies you can inject into Gmail's compose box
with one click - formal, friendly, or brief.

By default the AI runs ON YOUR COMPUTER. Email contents never
leave your laptop unless you explicitly switch to a cloud
provider in Settings.


WHAT YOU GET

* 8-category email badges (Meeting, Risk, External, Automated,
VIP, Follow-up, Newsletter, Finance) - multi-dimensional, not
a single "important" flag.

* Structured action items with type (Reply / Review / Schedule
/ Pay / Submit / Read), priority, and due dates - overdue
items highlighted.

* Pre-drafted replies in three tones - written in the same
language as the email, ready to inject into Gmail's compose
box.

* Three views: Inbox (flat list), Digest (grouped by category),
and Suggestions (AI-recommended actions for the day).

* Date-range fetching - last 24 hours through last month.

* 20 output languages.

* Source-quote highlighting - every AI claim traces back to a
verbatim excerpt from the email so you can audit it.

* In-memory only - nothing persisted to disk between launches.


POWERED BY SWARM-AI

Intelli Mailbox is built on Swarm-AI, IntelliSwarm's
open-source multi-agent framework for the JVM. Swarm-AI
provides 30+ pluggable tools (browser automation, email,
web search, code execution, file I/O, vector search), agent
orchestration with streaming tool-use events, 11 LLM provider
integrations (Ollama, OpenAI, Anthropic, Groq, Mistral,
DeepSeek, OpenRouter, Azure OpenAI, LM Studio, LocalAI, custom
OpenAI-compatible endpoints), and built-in prompt templates,
structured outputs, and an observability layer for tool-call
tracing.

Project home: https://intelliswarm.ai
Source code:  https://github.com/intelliswarm-ai


PRIVACY PROMISE

* Default AI runs locally via Ollama (privacy by default).
* Email contents never persisted to disk.
* No telemetry, no analytics, no phone-home.
* Cloud LLM providers are opt-in and clearly marked in the UI.
* Source code is auditable on GitHub.

SYSTEM REQUIREMENTS

* Windows 10 1809+ / 11 (or Windows Server 2019+).
* About 3 GB free disk (1 GB for the app + bundled Java
runtime, 2 GB for the AI model).
* 8 GB RAM recommended for smooth local AI on the default
model.
* Google Chrome (or Edge / Brave / any Chromium browser).
Intelli Mailbox attaches to it via the DevTools protocol to
read your Gmail. No Gmail API access required, no OAuth.

The installer ships its own Java 21 runtime. You don't need
to install Java separately. The AI engine (Ollama) is a
separate one-time install - the in-app Setup Check will guide
you through it on first launch.
